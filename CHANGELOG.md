# 更新日志 (CHANGELOG)

## [未发布] - 2025-10-14

### 新增功能
- ✨ **实时置信度显示**: 现在每条消息都会在控制台打印详细的检测结果
  - 显示用户信息
  - 显示消息内容预览（前100字符）
  - 显示是否判定为垃圾消息
  - **显示详细置信度**（百分比和小数形式）
  - 显示分类类型
  - 显示判定理由
  - 显示处理方式

### 改进
- 📊 增强日志输出：正常消息也会记录详细信息
- 🎨 美化控制台输出：使用分隔线和 emoji 图标
- 📝 跳过的消息也会显示基本信息

### 输出示例

```
================================================================================
📨 新消息检测
👤 用户: alice (ID: 123456789)
💬 内容: 大家好，很高兴认识大家！
🎯 垃圾消息判定: 否 ✅
📊 置信度: 12.50% (0.1250)
📋 类型: normal
💡 理由: 这是正常的问候消息，没有广告或恶意内容
🔧 处理: 保留
================================================================================

================================================================================
📨 新消息检测
👤 用户: spammer (ID: 987654321)
💬 内容: 加微信 xxx 免费领取资料，仅限今天！
🎯 垃圾消息判定: 是 ❌
📊 置信度: 95.00% (0.9500)
📋 类型: advertisement
💡 理由: 包含明显的商业广告和推广内容
🔧 处理: 删除+封禁
================================================================================
```
